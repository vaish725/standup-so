# standup-so

Turn your brain dump into a clean standup post in 10 seconds. Paste messy thoughts, get a structured Yesterday / Today / Blocked update ready to copy to Slack.

This repository contains the hackathon MVP described in `prd.md`.

Quick start (dev):

1. Install dependencies: npm install
2. Run dev server: npm run dev

For full details, see the PRD (`prd.md`).
